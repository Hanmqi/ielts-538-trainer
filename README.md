# IELTS 538 Daily Synonym Trainer

A small static PWA for IELTS reading synonym-replacement practice.

## Local Run

```powershell
py -m http.server 8765 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:8765/index.html
```

## Deploy

This project is static. It can be deployed to GitHub Pages, Cloudflare Pages,
Netlify, Vercel, or any HTTPS static hosting service.

For GitHub Pages, publish the repository root and open:

```text
https://<username>.github.io/<repository-name>/
```
